getKeySet(prefix, includeTimestamp)[source]¶

Get the set of keys for a given prefix.

CAVEAT: This method’s implementation is incomplete. If too short of a prefix is given, it will return nothing. includeTimestamp is also currently ignored.

Parameters:
  • prefix – The prefix to look under. Currently, this will be truncated to the current prefix length.
  • includeTimestamp – Currently Unused Return only keys modified after this timestamp.
Returns:

a dict containing all keys stored under the given prefix, with their timestamps.
