zht.node – ZHT Node Implementation¶

Node implementation. A node is the fundamental unit of the collective ZHT system.

class zht.node.Node(identity, repAddr, pubAddr, ctx=None, poolSize=200)[source]¶

Bases: object

Construct a new Node.

Parameters:
  • identity – The identity string of this Node.
  • repAddr – The ZMQ address to bind the REP socket to.
  • pubAddr – The ZMQ address to bind the PUB socket to.
  • ctx – The ZMQ Context object to operate from.
  • poolSize – The size of the greenlet pool this Node will operate from.

Connect to a Peer.

Parameters:addr – The ZMQ address of the Peer’s REP socket.
